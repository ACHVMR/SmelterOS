# Oracle Gateway (Boomerang Tool Suite)
# Unified API for Boomerang tools

This service provides a unified API for the Boomerang tool suite.

## Source Repos
- PPTist (Presentation Generation)
- gemini-cli (Google AI Terminal Integration)
- litellm-debugger (LLM Gateway & Routing)
- ghost-gcp-storage-adapter (Cloud Storage Adapter)

## Deployment
- Cloud Run: `oracle-gateway`
- Role: Tool dispatch, LLM routing, storage management
